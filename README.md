# Grade Calculator 🎓

## About This Project
This is my first Java project!
It calculates student grades based on marks of 5 subjects.

## What It Does
- Takes student name as input
- Takes marks of 5 subjects
- Calculates percentage
- Gives grade (A+, A, B, C, Fail)

## Technologies Used
- Java
- IntelliJ IDEA
- Git & GitHub

## What I Learned
- Scanner class for user input
- For loops
- If-else conditions
- Git commands
- How to push code to GitHub

## How to Run
1. Clone this repository
2. Open in IntelliJ IDEA
3. Run GradeCalculator.java
4. Enter student name
5. Enter marks of 5 subjects
6. See result!
